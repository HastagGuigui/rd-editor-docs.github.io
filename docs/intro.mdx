# Introduction

So you want to be a Rhythm Doctor? It's going to take an awful lot of practice.

-----

[Rhythm Doctor](https://store.steampowered.com/app/774181/Rhythm_Doctor/) has released on Steam, and includes the level editor. However, the free level editor is still available [here](https://giacomopc.itch.io/rdle).

<img className="left" src="img/download.png" alt="itch.io downloads page"/>

Some other useful links for making levels are:
- [Audacity](https://www.audacityteam.org/download/)
- [Anything that can calculate Beats per Minute, such as this one on All8](http://www.all8.com/tools/bpm.htm)
- [A list of animation easings](https://easings.net/)
- [youtube-dl, a tool that can download audio from many sources](https://ytdl-org.github.io/youtube-dl/index.html)
- [The Rhythm Doctor Discord](https://discord.gg/rhythmdr)
- [The list of custom levels](https://auburnsummer.github.io/rdlevels/)
