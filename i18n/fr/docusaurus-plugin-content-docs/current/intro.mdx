# Introduction

Voulez-vous devenir un Rhythm Doctor? Cela va demander énormément de pratique.

-----

[Rhythm Doctor](https://store.steampowered.com/app/774181/Rhythm_Doctor/) est sorti sur Steam, et inclut l'éditeur de niveau. Toutefois, la version gratuite est toujours disponible [ici](https://giacomopc.itch.io/rdle).

<figure>
  <img src="img/download.png" alt="my alt text"/>
  <figcaption>Cliquez sur ces boutons sur la page de l'éditeur gratuit pour le télécharger pour votre plateforme.</figcaption>
</figure>

D'autres liens qui pourraient vous être utile pour faire des niveaux sont:
- [Audacity](https://www.audacityteam.org/download/)
- [N'importe quel outil permettant de calculer les Battements par Minute, comme celui-ci sur All8](http://www.all8.com/tools/bpm.htm)
- [Une liste de courbes d'accélération d'animations](https://easings.net/fr)
- [youtube-dl, un outil permettant de télécharger de l'audio depuis plusieurs sources](https://ytdl-org.github.io/youtube-dl/index.html)
- [Le serveur Discord communautaire de Rhythm Doctor](https://discord.gg/rhythmdr)
- [La liste des niveaux personnalisés](rhythm.cafe/)
